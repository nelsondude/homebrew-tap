# homebrew-tap

Homebrew tap for [nelsondude](https://github.com/nelsondude) projects.

## Install

```bash
brew tap nelsondude/tap
```

## Available casks

| Cask | Description |
|------|-------------|
| `launchd-viz` | macOS Launch Agent & Daemon Manager |

```bash
brew install --cask launchd-viz
```

## How it works

Cask formulas are automatically updated when new releases are published in the upstream repos. No manual intervention needed.
